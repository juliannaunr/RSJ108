Julianna Fuentes's Language Learning Guide

An Instruction Challenge for JOUR 108

The complete guide for learning new languages, as well as some resources to make things easier!

Goal time: Practice at least 15 minutes a day
Awesome rating: 100% awesome

Many people choose to learn new languages for different reasons, but struggle to find good resources. Here are a few to help you get started.

  • Things You'll Need to Do

  • Learn the basics
  • Use language learning sites
  • Interact with others
  • Listen and watch
  • Read everything

  1. Learn the alphabet, a few common phrases, and proper pronounciation.

    This is necessary to create a good base for learning, and to make things easier in the long run.

  2. Try Duolingo, Babbel, Rosetta Stone, Memrise, and Conjuguemos.com

    These sites, plus others that can be found on the internet, are great for introducing new vocab and grammar.

    Tip: Start writing words you already know, and new ones each day!.

  3. Talk in the chosen language with either other learners, or native speakers.

    Get comfortable with speaking to others now, so that later it comes naturally. Plus, they can help you when necessary, and give you tips.

  4. Start easy. Watch cartoons, or movies you've seen before. Watch fun videos on YouTube, and eventually move onto shows and new movies.

    Take things one step at a time. As time goes on,expand your viewing experience. Be sure to watch with subtitles. You can start off with English subtitles, but be sure to change them to your target language as soon as you learn some basics to get used to spelling and vocab.

  5. From children's books, to instrcution labels, to documents — read it all.

    This is necessary for any language learner to be able to read, write, and spell properly in the language.
